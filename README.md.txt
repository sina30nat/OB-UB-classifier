# GP-Logit OB UB Classification Tool

This is a prototype Streamlit GUI for classifying underground drillholes as Balanced, Overbreak-Prone, or Underbreak-Prone using a trained GP-Logit (Genetic Programming + Logistic Regression) model.

## 📦 How to Run

1. Install requirements:

pip install -r requirements.txt

2. Launch the app:

streamlit run gui_classifier.py

## 📁 Files

- `gui_classifier.py` – Streamlit GUI script
- `models/` – contains `logistic_regression_tuned_fitted.joblib`
- `requirements.txt` – lists required Python libraries
